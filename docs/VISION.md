# Vision

Project Atabey treats a cell track as a living transition through time, not only a
set of independent detections.

The framework is intentionally modest. It should produce valid Kaggle outputs, but
its research value is in how it preserves lineage, uncertainty, transient weak
signals, and state changes across 3D+time embryo videos.

Atabey borrows from Lumina only at the mechanism level:

- memory traces become track-local history
- dormant potential becomes latent candidate retention
- local pressure becomes crowding and motion conflict
- phase transitions become explicit cell-state transitions
- reflective interpretation becomes review-oriented uncertainty notes

Boundary: these mechanisms must never be described as biological certainty. They are
bounded tracking aids.
